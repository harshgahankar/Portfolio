# Harsh Gahankar Portfolio

A modern, interactive personal portfolio website designed and developed using **HTML, CSS, and JavaScript**.  
This portfolio showcases my projects, skills, experiences, milestones, achievements, and contact details with a creative dark-orange visual theme.

## Live Demo

🔗 **Portfolio:** https://harsh-gahankar.vercel.app

## About

This portfolio represents my journey as a **Computer Engineering student, developer, graphic designer, and creative tech enthusiast**.  
The design focuses on bold typography, interactive sections, smooth animations, and a premium visual experience.

## Features

- Interactive hero section with 3D moon
- About Me section with hover photo stack
- Drag-based “Explore My Personality” section
- Horizontal scroll Projects showcase
- Filter-based Skills section
  - Software skills
  - Designing skills
- Zigzag Experience timeline
- Milestones & Wins section
- Final contact section with social links
- Resume download button
- Fully responsive layout
- Dark theme with orange accent styling

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js
- Vercel
